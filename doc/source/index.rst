欢迎来到 sokort 文档
==================================

封装中国气象局数值预报中心数值预报业务系统使用的图片制作脚本。

仅支持使用 NCL 绘图脚本，目前已支持的数值预报业务系统如下：

- GRAPES 全球预报系统
- GRAPES 区域预报系统
- GRAPES 区域台风预报系统
